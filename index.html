🆔 IP (endereço)

É como o endereço da sua casa. Cada dispositivo tem um IP único.

📌 Cliente: 192.168.0.100

📌 Servidor: 200.100.50.10

⬆️ Os pacotes carregam esses endereços.

📇 DNS (tradutor)

Você digita www.google.com — o DNS descobre o IP.

🌐 "google.com" → 172.217.29.132

📦 Pacotes divididos

Os dados são quebrados em pedaços. Cada pedaço (pacote) pode seguir um caminho diferente!

🔄 Na animação, pacotes azuis (pedidos) e verdes (respostas) usam rotas variadas.

⚡ Roteadores encaminham cada pacote individualmente ⚡
